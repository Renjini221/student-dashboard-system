# Student Dashboard App

this is my student dashboard project
it is a simple web app to manage student details like subjects, marks and other information in one place

## what it does

* store student information
* track marks
* view subjects
* show everything in a single dashboard

## tech used

* html
* css
* javascript
* flask (backend)

## why i made this

managing student data in different places can be messy
so i built this to keep everything organized and easy to access

## features

* clean ui
* easy to use
* organized data
* lightweight

## what i learned

* project structure
* frontend and backend basics
* handling data

## future plans

* add login system
* improve ui
* add charts for marks
* more features
